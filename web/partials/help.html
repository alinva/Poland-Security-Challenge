Briefing

  • We will introduce you to some Security concepts like Brute-force, XSS, Path Traversal and more.

  • For further learning go to OWASP - "The Open Web Application Security Project",
    an online community which creates freely-available articles, methodologies, documentation, tools, and technologies in the field of web application security.


    • Instruction page

    Goal

  • Your target is to hack the car and the obstacles along the way in order to arrive to a very important SAP yearly board meeting - you have only 20 minutes!

    • Useful Hints

  • The only requirement is common sense.
  • In Chrome, open the developer tools with F12, CTRL+SHIFT +F.
  • You can try right-click and inspect elements.
  • The network tab can also be useful.

  • Remember to do it in chrome incognito so as to not leaving any cookies.

    • Technologies

    We used MEAN (Mongo, Express, Angular, Node) stack

  • MongoDB is the persistence layer.
  • ExpressJS is the backend layer for building the web application.
  • AngularJS is the frontend layer, gives us the MVC in the client side.
  • NodeJS runs our JavaScript server.
